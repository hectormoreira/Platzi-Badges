# Platzi-Badges
Course React.js - Platzi: PlatziBadges

## Global create-react-app
```
npm install -g create-react-app
```

## Install Node Modules
```
cd Platzi-Bages
npm install
```

## Install React Router (Browser)
```
npm install react-router-dom
```

## Install Bootstrap
```
npm install bootstrap
```

## Run server
```
npm run start
```
